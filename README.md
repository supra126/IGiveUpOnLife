# 不想努力了 I Give Up On Life

> 打造完整品牌視覺資產？不用，我不想努力了。

一個由 AI 驅動的產品行銷內容設計工具，讓 **小GG（AI 總監）** 幫你結合產品識別、品牌故事與競品策略，生成專業的廣告海報概念與視覺素材。

## ✨ 功能特色

### 🎯 Phase 1: 視覺策略選擇（秒速決策）

- **智能產品分析**：上傳產品圖片，AI 自動解讀視覺特徵與品牌語意
- **三條策略路線**：即時生成 3 種截然不同的行銷策略方向（純文字策略卡片）
- **詳細策略資訊**：每條路線包含主打標語、視覺風格描述、目標受眾定位
- **快速響應**：無需等待圖片生成，3 秒內完成策略分析

### 🚀 Phase 2: 完整內容企劃生成（自動觸發）

#### 📐 多平台尺寸支援
選擇策略路線後，可自由選擇需要的圖片尺寸：
- **1:1 方形圖** - 適合 FB 貼文、IG 輪播、電商主圖
- **9:16 直式長圖** - 適合限時動態、Instagram Stories、Reels
- **4:5 直式圖** - 適合 IG Feed 主頁、優化手機瀏覽
- **16:9 橫式長圖** - 適合封面、廣告圖片、橫幅設計
- **1:1 商業攝影** - 專業商品攝影風格（工作室燈光、高端相機質感）

#### 📝 內容企劃功能
- **每個尺寸 3 組方案**：AI 自動生成不同行銷角度的內容方案
  - 第 1 組：功能導向（強調產品功能、特點）
  - 第 2 組：情感導向（強調生活情境、感受）
  - 第 3 組：數據/背書導向（強調成效、評價、認證）
- **可編輯腳本**：所有生成的文案與視覺提示詞都可即時編輯調整
- **雙模式切換**：
  - 🔍 **內容來嘴看看**（腳本審閱模式）- 編輯標題、文案、視覺構圖、AI Prompt
  - 🎨 **爆肝產圖**（圖片製作模式）- 實際生成視覺素材

### 🎨 專業圖片生成功能

#### 提詞版 & 參考版
每個方案都支援兩種生成模式：
- **提詞版**：使用 AI 自動生成的視覺 Prompt 創作
- **參考版**：上傳參考圖片，控制相似度生成
  - 低相似度：創意發揮（僅參考色調與氛圍）
  - 中等相似度：適度參考（匹配光線與色彩，構圖可變）
  - 高相似度：完全模仿（匹配構圖、佈局、風格）

#### 文字與視覺控制
- ✅ **顯示內容選項**：可選擇在圖片上疊加標題和文案
- ✅ **字體粗細選擇**：使用開源 **思源黑體（Noto Sans TC）**，支援 4 種字重
  - Regular (400) - 正常粗細
  - Medium (500) - 適中粗細
  - Bold (700) - 粗體
  - Black (900) - 特粗體
- ✅ **品牌 Logo 上傳**：自動放置在四個角落之一
- ✅ **即時編輯**：直接在產圖模式修改標題和文案
- ✅ **獨立控制**：每張圖片可以單獨設定是否顯示文字和字重

#### 圖片操作
- 📥 **下載功能**：滑鼠移至圖片上方顯示下載按鈕
- 🔄 **重繪功能**：不滿意可以重新生成
- 💾 **持久化設定**：上傳的產品圖和 Logo 在所有方案間共用

### 📊 專業報告輸出

- **策略報告下載**：完整的產品分析、策略選擇與內容企劃 (.txt)
- **視覺素材下載**：所有生成的圖片可單獨下載保存

## 🛠️ 技術棧

- **前端框架**: React 19 + TypeScript
- **構建工具**: Vite 6
- **AI 模型**: Google Gemini
  - `gemini-2.5-flash` - 文字分析與內容規劃
  - `gemini-3-pro-image-preview` - 專業圖片生成
- **樣式**: Tailwind CSS
- **字體**:
  - 介面：Inter + Playfair Display
  - 生成圖片：Noto Sans TC（思源黑體）

## 📦 安裝與使用

### 環境要求

- Node.js 18+
- npm 或 yarn

### 安裝步驟

```bash
# 克隆專案
git clone https://github.com/mag477/IGiveUpOnLife.git

# 進入專案目錄
cd IGiveUpOnLife

# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev
```

### 配置 API Key

1. 點擊右上角的「API 設定」按鈕
2. 輸入您的 Google Gemini API Key
3. API Key 會安全地儲存在瀏覽器本地存儲中

> 💡 如何獲取 Gemini API Key：前往 [Google AI Studio](https://aistudio.google.com/app/apikey) 註冊並創建 API Key

## 🎨 使用流程

### 方案 A：策略優先流程（當前版本 v3.0）

#### 1. 一次填完所有資訊

- 上傳產品圖片（支援 JPG、PNG）
- 填寫產品名稱、產品資訊、產品網址（可選）
- 填寫參考文案 / 競品參考（可選）

#### 2. AI 策略分析

- 點擊「啟動努力」按鈕
- 等待 AI 分析產品（約 3-5 秒）

#### 3. 選擇視覺策略路線

- 查看 3 條不同的策略路線（純文字）
- 每條路線包含：主打標語、視覺風格、目標受眾
- 點選一條路線

#### 4. 選擇圖片尺寸

系統會自動進入尺寸選擇畫面，勾選你需要的尺寸：
- ✅ FB 貼文 (1:1 方形圖)
- ✅ 限時動態 (9:16 直式長圖)
- ✅ IG (4:5 直式圖)
- ✅ 橫式貼文 (16:9 橫式長圖)
- ✅ 商業攝影 (1:1 專業商品攝影)

#### 5. 自動生成內容腳本

- 點擊「開始生成內容」
- AI 為每個選定的尺寸生成 3 組不同的內容方案
- 完成後自動滾動到內容區域

#### 6. 上傳產品圖與 Logo（選填）

在「爆肝產圖設定」區域：
- 📸 上傳產品圖片（必要，用於圖片生成）
- 🏷️ 上傳品牌 Logo（選填，會自動放置在圖片角落）

#### 7. 腳本審閱與編輯（內容來嘴看看）

切換至「內容來嘴看看」模式：
- 查看所有方案的標題、文案、視覺摘要
- 編輯任何不滿意的文案
- 調整 AI Prompt（進階功能）
- 可為每個方案重新生成 Prompt

#### 8. 圖片製作（爆肝產圖）

切換至「爆肝產圖」模式，針對每個方案：

##### 基本設定
1. **勾選「顯示內容（標題 + 文案）」** - 決定是否在圖片上疊加文字
2. **編輯標題和文案** - 可直接在產圖模式修改
3. **選擇字體粗細** - 標題和文案各有下拉選單
   - Regular - 正常粗細
   - Medium - 適中粗細
   - Bold - 粗體（標題預設）
   - Black - 特粗體

##### 選擇生成模式
- **提詞版**（預設）
  - 使用 AI 自動生成的視覺 Prompt
  - 自動加入產品圖和 Logo（如果有上傳）

- **參考版**
  - 上傳參考圖片
  - 選擇相似度：低/中/高
  - 同樣會加入產品圖、Logo 和文字

##### 生成與下載
1. 點擊播放按鈕開始生成
2. 等待約 10-20 秒（視尺寸而定）
3. 滑鼠移至圖片上方：
   - 📥 下載按鈕 - 下載圖片
   - 🔄 重繪按鈕 - 重新生成

#### 9. 匯出報告

點擊「下載策略報告」按鈕：
- 匯出完整的產品分析
- 包含所有方案的標題、文案、視覺摘要、Prompt

## 🌟 核心特點

### 使用者體驗
- **零設計門檻**：不需要設計經驗，AI 自動生成專業視覺策略
- **極速響應**：策略決策只需 3 秒，無需等待圖片生成
- **流暢體驗**：精心優化的動畫與 UI，提供愉悅的使用體驗
- **完全可控**：所有 AI 生成內容都可編輯，保持創意自主權

### 行銷專業
- **品牌一致性**：基於品牌故事與產品特性，確保視覺風格統一
- **競品洞察**：可選擇性分析競品文案，學習成功的說服邏輯
- **多角度策略**：每個尺寸提供功能、情感、數據三種行銷角度
- **多平台支援**：一次生成所有社群平台需要的尺寸

### 技術優勢
- **成本優化**：只生成用戶真正需要的圖片，節省 API 成本
- **開源字體**：使用思源黑體（Noto Sans TC），無版權問題
- **彈性生成**：支援提詞版和參考版雙模式
- **持久化設計**：上傳的圖片和設定在所有方案間共用
